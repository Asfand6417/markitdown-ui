# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of MarkItDown Converter UI
- Web interface for document conversion
- Support for PDF, DOCX, PPTX, XLSX, HTML, Images, TXT
- Drag and drop file upload
- Real-time conversion feedback
- Docker support
- GitHub Pages documentation
- Comprehensive README and guides
- Contributing guidelines
- Issue templates
- Code of Conduct
- Security policy

### Features
- Beautiful, responsive UI
- Multiple file format support
- One-click markdown download
- Local file processing (privacy-focused)
- GitHub Actions CI/CD workflow
- Branch protection rules
- Automated testing

### Documentation
- Setup instructions
- Deployment guides (Docker, Heroku, Vercel)
- API documentation
- Contributing guide
- FAQ section
